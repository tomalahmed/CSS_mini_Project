A simple landing page with a toggleable sidebar menu, built with vanilla HTML/CSS and Font Awesome.

## Project Structure
- [index.html](index.html): Page markup, sidebar toggle logic via checkbox, menu links, and footer.
- [styles.css](styles.css): Layout, background image, sidebar animation, typography, and icon styling.
- Resource/: Place static assets (e.g., background image `mango.jpg`).

## Getting Started
1. Ensure `Resource/mango.jpg` exists to show the background.
2. Open `index.html` in a browser (no build step required).

## Customization
- Update colors, spacing, and hover effects in [styles.css](styles.css).
- Replace menu labels/links inside the sidebar in [index.html](index.html).

## External Dependencies
- Google Font: Montserrat (loaded in [index.html](index.html)).
- Font Awesome 6.4.0 CDN for icons.
